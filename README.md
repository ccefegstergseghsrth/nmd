# 长安志  数字文化展馆

一个展示长安（西安）历史文化的数字展馆网站。

## 项目结构

`
 index.html          # 卷首页面
 culture.html        # 文化解码首页
 culture-detail.html # 文化解码详情页
 visit.html          # 探访实录首页
 visit-detail.html   # 探访实录详情页
 archives.html       # 风物档案首页
 archives-detail.html # 风物档案详情页
 map.html            # 长安舆图页面
 css/                # 样式文件
 js/                 # JavaScript文件
 data/               # 数据文件
 images/             # 图片资源
`

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5

## 本地运行

直接打开 index.html 即可浏览。

## 部署

支持 GitHub Pages 部署。
